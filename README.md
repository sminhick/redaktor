Здесь присутствуют три модуля, с помощью которых был создан редактор:
1.1 Модуль colorPicker отвечает за палитру цветов для отображения фигур.
1.2 Модуль imageProcessor содержит в себе описание функций, методов, преображений, с помощью которых можно изменить картинку.
1.3 Модуль imageWork отвечает за файлы, фотографии
